# SPORT-SPECIFIC FITNESS PERFORMANCE APPLICATION

# OVERVIEW
A full-stack application that provides sport-specific training plans for amateur athletes. athelets 
are able to choose a sport, a training goal, follow structured workouts, log
sessions, and track progress on a visual dashboard.

# REQUIREMENTS
To make sure no issues accure make sure you have the following installed before running:

- Node.js v18 or higher (https://nodejs.org)
- npm (comes with Node.js)
- A modern web browser (Chrome, Firefox, Edge)
- An internet connection (required for Supabase database)

# RUNNING THE APPLICATION

STEP 1 — Start the Backend Server Open a terminal and run the following commands:
    - cd server
    - npm install
    - npm run dev
    
You should see:
    Server running on port 5000

# Keep this terminal open.

STEP 2 — Start the Frontend Open a SECOND terminal and run:
    - cd client
    - npm install
    - npm run dev

You should see:
    Local: http://localhost:5173

Keep this terminal open.

STEP 3 — Open the App You should see the login page.

# USING THE APPLICATION

1. SIGN UP
2. DISCLAIMER
3. SELECT A SPORT
4. SELECT A GOAL
5. VIEW WORKOUTS
6. VIEW PROGRESS
